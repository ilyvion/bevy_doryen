fn on_startup(self) -> SystemSetConfigs

👎Deprecated since 0.11.0: app.configure_sets((A, B).on_startup()) has been deprecated in favor of app.configure_sets(Startup, (A, B)). Please migrate to that API.

This used to configure the sets in the CoreSchedule::Startup schedule. This was a shorthand for self.in_schedule(CoreSchedule::Startup).

Panics

Always panics. Please migrate to the new App::configure_sets with the Startup schedule: Ex: app.configure_sets((A, B).on_startup()) -> app.configure_sets(Startup, (A, B))

fn in_schedule(self, _schedule: impl ScheduleLabel) -> SystemSetConfigs
