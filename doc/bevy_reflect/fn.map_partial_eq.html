Function bevy_reflect::map_partial_eq

pub fn map_partial_eq<M: Map>(a: &M, b: &dyn Reflect) -> Option<bool>
Expand description

Compares a Map with a Reflect value.

Returns true if and only if all of the following are true:

  • b is a map;
  • b is the same length as a;
  • For each key-value pair in a, b contains a value for the given key, and Reflect::reflect_partial_eq returns Some(true) for the two values.

Returns None if the comparison couldn’t even be performed.
