pub struct WebGLRenderingContext {

Expand description

The OpenGL rendering context. This is the struct providing most of the OpenGL API.

impl WebGLRenderingContext

source

pub fn new<'p>(loadfn: WebGLContext<'p>) -> WebGLRenderingContext

create an OpenGL context.

uni-gl should be used with the uni-app crate. You can create a WebGLRenderingContext with following code :

ⓘ
let app = uni_app::App::new(...);
let gl = uni_gl::WebGLRenderingContext::new(app.canvas());
